# JourneyNote ✨

JourneyNote is a simple and elegant web app to capture your thoughts, notes, and personal journey.  
Deployed live at: [journey-note.netlify.app](https://journey-note.netlify.app/)

---

## 🚀 Features
- Create and manage personal notes
- Clean and responsive UI
- Fast and secure deployment with Netlify
- Built with modern web technologies

---

## 🛠️ Tech Stack
- **Frontend:** React (with hooks & functional components)
- **Styling:** CSS 
- **Deployment:** Netlify

---

## 📦 Installation & Setup

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/journey-note.git
cd journey-note
npm install
npm start
